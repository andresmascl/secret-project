# Repository Structure - Quick Reference

This document provides a quick overview of the complete repository structure for Scrap AI Bot.

## Root Level Files

```
scrap-ai-bot/
├── README.md                 # Main user documentation
├── ARCHITECTURE.md           # Technical architecture for developers
├── AI_INSTRUCTIONS.md        # Guidelines for AI coding assistants
├── CONTRIBUTING.md           # How to contribute
├── CHANGELOG.md              # Version history and changes
├── LICENSE                   # AGPL-3.0 license
├── .env.example              # Environment variables template
├── .gitignore               # Git ignore patterns
├── .gitmodules              # Submodule configuration
├── requirements.txt         # Python production dependencies
└── requirements-dev.txt     # Python development dependencies
```
